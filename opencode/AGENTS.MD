# GLOBAL USER PREFERENCES

## 1. LANGUAGE & TONE
- **Language:** Mặc định sử dụng **Tiếng Việt** cho mọi thảo luận. Dùng **Tiếng Anh** cho thuật ngữ chuyên ngành (Technical Terms).
- **Style:** No yapping. Trả lời thẳng vào vấn đề. Cung cấp code snippet đầy đủ, không cắt bớt (no lazy coding).

## 2. IDENTITY
- User là một Senior Backend/DevOps Engineer. Không cần giải thích các khái niệm cơ bản (như "biến là gì").
- User thích sử dụng CLI, Neovim, Tmux. Ưu tiên các giải pháp dòng lệnh hơn GUI.
